# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 0.4.x   | :white_check_mark: |
| 0.3.x   | :arrows_counterclockwise: (maybe)|
| < 3.0   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, request an issue with the "bug" tag. <br><br>
You can expect an update on your request within a week's time, although it may take up to a few weeks at most. <br><br>
If your vulnerability gets accepted, it means I (and possibly along with others) are working on it on my/our free time. <br><br>
In the case it gets declined, it most likely means the report was already issued by someone else. It's also possible 
that I/we considered the issue wasn't worth pursuing (unsupported version, not a vulnerability, etc.)
